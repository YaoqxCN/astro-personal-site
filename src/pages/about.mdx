### 关于网站
**这个小网站是怎么建成的？**\
我曾经考虑过搭建一个属于自己的个人网站，毕竟那是一件很酷的事。但那时的我只会 Python 和 C++，你说我该怎么办？\
我开始寻找一些建站工具，大多都是网站工具——那时的我完全不了解前端。直到后来我看到了别人的网站，很显然他们并没有用建站工具。\
于是，我开始学习 HTML 和 CSS，尝试自己写一个网站。很快，我就写出了一个简单的网站，但过不了几天，我就嫌它丑，所以我又重写了一个。但还是丑。\
虽然 HTML 和 CSS 十分简单的东西，但写起来很麻烦，而我又是一个很懒的人，不想写了，于是又开始寻找建站工具。\
但这次不一样，我了解了前端，知道了很多框架之类的东西。接着，我就找到了一样东西：Astro。
我当然只是在 Astro 上找模板主题，很快我就找到了一个，也就是现在这个。
我对这个主题很满意，就 git clone 下来，稍微做修改内容，用了仅仅一天就搭建完成了，真的是懒人的福音。
